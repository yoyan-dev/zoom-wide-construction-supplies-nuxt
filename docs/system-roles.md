Role Description Permissions

Admin:
System owner or head
Full access: manage users, categories, products, suppliers, orders, deliveries, inventory, payments, system settings

Manager / Staff:
Day-to-day operations Manage orders, carts, deliveries, view products, inventory updates

Customer Buyers / clients:
Browse products, add to cart, place orders, track deliveries, make payments

Warehouse / Inventory Manager:
Stock control Manage inventory, update stock, view products and deliveries

Finance / Accountant:
Payment and finance Manage payments, generate reports, view orders, cannot modify products

Driver / Delivery Staff:
Delivery handling Update delivery status, view assigned orders
Supplier Material suppliers Optional portal to update product supply, stock availability
Auditor Internal/external auditing View-only access to orders, inventory, payments; cannot modify
